# Stickman Arena

A browser-based stickman fighting game with multiple biomes, weapons, and animals.

## How to Play

1. Open `game.html` in any modern web browser
2. Use **WASD** or **Arrow Keys** to move
3. Press **Space** to jump
4. Press **Z** or **J** to attack
5. Pick up weapons for extra damage!
